# Manuscript Options

All available options for `manuscript` projects are documented below. See [Quarto Manuscripts](../../../docs/manuscripts/) for an in-depth guide to creating manuscripts with Quarto.

|  |  |
|:---|:---|
| `article` | The input document that will serve as the root document for this manuscript |
| `code-links` | Code links to display for this manuscript. |
| `manuscript-url` | The deployed url for this manuscript |
| `meca-bundle` | Whether to generate a MECA bundle for this manuscript |
| `notebooks` | Options for notebooks included under the heading “Notebooks”. See [Including Notebooks](../../../docs/manuscripts/components.llms.md#including-notebooks) for more details. |
| `resources` | Additional file resources to be copied to output directory |
| `environment` | Files that specify the execution environment (e.g. renv.lock, requirements.text, etc…) |
