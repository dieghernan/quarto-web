# Quarto 1.6 Pre-Release

Published

March 3, 2026

Quarto 1.6 includes the following new features:

- [Support for **brand.yml**](../../../docs/authoring/brand.llms.md): Customize the appearance of your Quarto documents across formats with a single `_brand.yml` file.

- Reveal JS updated to v5.1.0. Adds:

  - [Jump To Slide](../../../docs/presentations/revealjs/presenting.llms.md#jump-to-slide): a menu to quickly navigate between slides.
  - [Scroll View](../../../docs/presentations/revealjs/presenting.llms.md#scroll-view): a mode to view a presentation as a scrollable page.

- [`{{< contents >}}` shortcode](../../../docs/authoring/contents.llms.md): Flexibly rearrange the contents of your document, including moving the results of an executable code cell.

- [Landscape mode](../../../docs/authoring/article-layout.llms.md#landscape-mode) blocks: Use fenced divs with class `.landscape` to set portions of `docx`, `pdf` and `typst` documents to landscape. Thanks to [`@edvinsyk`](https://github.com/edvinsyk)!

See the [Pre-Release Download Page](../../../docs/download/prerelease) for a complete list of all the changes and bug fixes.
