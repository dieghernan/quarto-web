# Quarto 1.3 Pre-Release

Published

March 3, 2026

Quarto 1.3 includes the following new features:

- [Confluence Publishing](../../../docs/publishing/confluence.llms.md)—Publish quarto documents and projects to Confluence spaces.

- [Multi-Format Publishing](../../../docs/output-formats/html-multi-format.llms.md)—Simple discovery of other formats for HTML documents.

- [Jupyter Cell Embedding](../../../docs/authoring/notebook-embed.llms.md)—Embed code and outputs from Jupyter Notebooks in Quarto documents.

- [Article Grid Customization](../../../docs/output-formats/page-layout.llms.md#grid-customization)—Customize the widths within the article grid in HTML documents.

- [Code Block Annotation](../../../docs/authoring/code-annotation.llms.md)—Use annotation to provide contextual comments to help readers understand code.

- [Quarto Book AsciiDoc Support](../../../docs/books/book-output.llms.md#asciidoc-books)—Output Quarto books to AsciiDoc files

- [Website Navigation Improvements](../../../docs/prerelease/1.3/website-nav.llms.md)—Improved navigation for Quarto websites.

- [Mermaid Diagram Theming](../../../docs/authoring/diagrams.llms.md#mermaid-theming)—Mermaid diagram’s appearance automatically matches your document’s theme (or customize it)

- [PDF: SVG and Remote Images](../../../docs/prerelease/1.3/pdf.llms.md)—Improved handling for SVG images and remote images in PDF documents.

- [`kbd` Shortcode](../../../docs/authoring/markdown-basics.llms.md#keyboard-shortcuts)—Show well formatted keyboard shortcuts in Quarto documents.

- [Extensions: Quarto AST](../../../docs/prerelease/1.3/ast.llms.md)—Use custom AST nodes for Quarto types like Callout when writing extensions.

- [Extensions: HTML Tables](../../../docs/prerelease/1.3/tables.llms.md)—HTML tables are now processed and accessible to Lua filters in all formats (not just HTML).

See the [Pre-Release Download Page](../../../docs/download/prerelease.llms.md) for a complete list of all the changes and bug fixes.
