In this video, I walk you through creating documents, presentations, and websites and publishing with Quarto. The video features authoring Quarto documents with executable R code chunks using the [RStudio Visual Editor](../../../../docs/visual-editor/index.llms.md).

Select highlights include:

- Inserting [cross references](../../../../docs/authoring/cross-references.llms.md) to tables and figures
- Adding a [citation](../../../../docs/visual-editor/technical.llms.md#citations) from a DOI
- Seamlessly switching between output formats as well as creating [multi-format documents](../../../../docs/output-formats/html-multi-format.llms.md)
- Customizing the [output location](../../../../docs/presentations/revealjs/#output-location) of code in presentations
- Creating a [website](../../../../docs/websites/index.llms.md) from scratch
- Publishing the website to [QuartoPub](../../../../docs/publishing/quarto-pub.llms.md)

If you would like to follow along as you watch the video, you can find the source code for everything created in the video in [this GitHub repository](https://github.com/mine-cetinkaya-rundel/get-started-quarto) and the published [website on QuartoPub](https://mine.quarto.pub/welcome-to-quarto/).
